Frequently Asked Questions (FAQ)

  1. Is it possible to download a file with an image of the table I have created?

    No, currently it is not possible.

    However, if you are creating an HTML table you may use Preview to show a preview of your table. Next, you may take out a screenshot and cut the table using a graphics editing program.

  2. Tables Generator does not seem to do anything

    Please, try to use the latest Chrome (or Chromium) browser. Firefox should be OK, too. The generator may not work properly with any other browser or on mobile phones and tablets. We are sorry for the inconvenience.

  3. Is the code of the Tables Generator open source?

    No, not yet. Although we do not rule out such possibility in the future.

  4. Do you provide an open API to the Tables Generator?

    No.
